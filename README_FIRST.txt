Instructions for the   challenge app.


 1. Extract the files   Change the parameters of dbname ,username and password  on .env file
 2. Seed the DB files into the DB
 3. Run composer or  in the  cli  php artisan serve  
 5. Tasks 1_ are visible by  entering  http://127.0.0.1:8000  the implementations are made by using blade pages and UserController
 6. Tasks 2_ you can try http://127.0.0.1:8000/transactions?source=db  or  http://127.0.0.1:8000/transactions?source=csv because the transactions.csv is on the public folder.
    The implementation is made in the LogController
